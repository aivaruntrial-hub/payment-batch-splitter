Payment Batch Splitter

Drop in a bank payment batch PDF (e.g. a Kotak NEFT/RTGS advice batch) and this tool splits it into one PDF per beneficiary, named "Beneficiary Name_Amount Paid.pdf" using the details on each voucher.

🔒 Everything runs locally in this browser tab. Your file is never uploaded to any server — splitting and name extraction all happen on your own device.
